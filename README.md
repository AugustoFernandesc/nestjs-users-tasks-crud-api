# NestJS Users & Tasks CRUD API

API REST desenvolvida como exercício técnico utilizando NestJS, TypeORM e PostgreSQL.  
O sistema permite o gerenciamento de usuários e suas respectivas tarefas.

## Tecnologias utilizadas

- NestJS
- TypeScript
- Node.js
- TypeORM
- PostgreSQL
- class-validator

## Funcionalidades

### Users
- Criar usuário
- Listar usuários
- Buscar usuário por ID
- Atualizar usuário
- Remover usuário

### Tasks
- Criar tarefa
- Listar tarefas
- Buscar tarefa por ID
- Atualizar tarefa
- Remover tarefa
